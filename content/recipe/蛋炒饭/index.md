## 食材

> 2 人份

| 材         | 量        | 备注   |
| :--------- | :-------- | :----- |
| 米         | 150g      |        |
| 鸡蛋       | 4 ～ 6 个 |        |
| 牛肉/鸡肉  | 250g      | 可不放 |
| 洋葱       | 小半个    | 可不放 |
| 冷冻蔬菜粒 | 100g      |        |
| 料酒       | 少量      |        |
| 酱油       | 少量      |        |
| 胡椒粉     | 少量      |        |
| 盐         | 少量      |        |

## 步骤

1. 电饭煲烧白米饭<em>（比平时烧饭加的水少一点，让饭粒硬一些）</em>
2. 肉切丁，鸡蛋打散加料酒加盐，洋葱切丁
3. 炒熟鸡蛋<em>（嫩一点）</em>，盛出来备用
4. 少量油炒肉，加料酒，加少量盐给肉调味，不用炒熟
5. 加洋葱丁、蔬菜粒，翻炒
6. 加一点点水，盖上锅盖，烧 5-10 分钟
7. 开盖后，加入之前炒熟的蛋，加入米饭，加酱油，翻炒均匀

## 备注

- 炒肉和蔬菜的步骤大约 20 分钟
  - 用电饭煲烧饭的同时，可以备菜和炒菜
  - 白米饭烧好后刚好可以放进锅里和菜一起翻炒
- 肉丁切小一些可以减少盖锅盖烧的时间
