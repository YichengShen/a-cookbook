## 食材

> 1 人份

| 材料     | 量     | 备注               |
| :------- | :----- | :----------------- |
| 切片面包 | 2 片   |                    |
| 牛油果   | 0.5 个 |                    |
| 鸡蛋     | 1 个   |                    |
| 番茄     | 0.2 个 | 用来解腻（可不放） |
| 黄油     | 一小块 | 多一点会比较香     |
| 胡椒     | 少量   |                    |
| 盐       | 少量   |                    |

## 步骤

1. 牛油果切小块，捣成泥（备用）
2. 番茄切片（备用）
3. 煎蛋，上面撒盐、胡椒
4. 用不粘锅，小火融化黄油，然后放入切片面包
5. 翻面，让双面都吸收到黄油
6. <u>小火</u>煎至面包酥脆，出锅
7. 把牛油果涂在面包上，放上煎蛋、番茄

## 备注

- 选偏熟的牛油果
